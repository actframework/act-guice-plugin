# act-guice-plugin

Implement JSR 330 based on Google Guice for ActFramework

## Usage

In your `pom.xml` add the following dependency:

```
    <dependency>
      <groupId>org.actframework</groupId>
      <artifactId>act-guice</artifactId>
      <version>0.1.1-SNAPSHOT</version>
    </dependency>
```
